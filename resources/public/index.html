    • About creepMiner

    The creepMiner is an open source client application for mining Burst on a pool or solo, authored by Creepsky. You can access the project on Github:

    • Code
    • Releases

    If you find any bugs or encounter problems while using creepMiner, please open a github issue explaining your problem.

    For minor problems, like configuration issues, or if you would just like to chat with us, you can also join our Discord.

    Donate

    While creepMiner is an open source project that can be used free of any charge, donations are very appreciated.
