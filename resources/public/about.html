    • About creepMiner

    The creepMiner is an open source client application for mining Burst on a pool or solo, authored by Creepsky. You can access the project on Github:

    • Code: https://github.com/Creepsky/creepMiner

    • Releases: https://github.com/Creepsky/creepMiner/releases

    If you find any bugs or encounter problems while using creepMiner, please open an issue explaining your problem here:

    • Issues: https://github.com/Creepsky/creepMiner/issues
